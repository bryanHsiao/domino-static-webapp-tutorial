從 Actions Artifact 更新 `updatesite.nsf`Updating `updatesite.nsf` From Actions Artifacts

現在 GitHub Actions 已經會自動 build 專案並產出 artifact。 這份頁面說明如何把 CI 產物拿下來，匯入 Domino 的 `updatesite.nsf`， 然後完成 HTTP reload 與站台驗證。 GitHub Actions now builds the project automatically and publishes artifacts. This page explains how to download those CI outputs, import them into Domino `updatesite.nsf`, and complete the HTTP reload and verification steps.

你真正需要的 artifactWhich Artifact You Actually Need

更新 `updatesite.nsf` 時，真正需要的是 `update-site-repository`。 這個 artifact 對應的是 `hello-webapp/target/repository/`，裡面會包含： `site.xml`、`artifacts.jar`、`content.jar`、`features/` 與 `plugins/`。 To update `updatesite.nsf`, the artifact you actually need is `update-site-repository`. It corresponds to `hello-webapp/target/repository/` and contains `site.xml`, `artifacts.jar`, `content.jar`, `features/`, and `plugins/`.

做法 A：從 GitHub 網頁下載Option A: Download From GitHub Web UI

  1. 打開 repo 的 `Actions` 頁。Open the repository `Actions` tab.
  2. 打開最新一筆成功的 `Build Tutorial Site`。Open the latest successful `Build Tutorial Site` run.
  3. 在 `Artifacts` 區塊下載 `update-site-repository`。Download `update-site-repository` from the `Artifacts` section.
  4. 解壓縮並確認裡面有 `site.xml`。Extract it and confirm that `site.xml` is present.

做法 B：用 GitHub CLI 下載Option B: Download With GitHub CLI

gh run download 24794809561 `
  --repo bryanHsiao/domino-static-webapp-tutorial `
  --name update-site-repository `
  --dir artifacts/update-site-repository

下載後要匯入的檔案是 `artifacts/update-site-repository/site.xml`。 After download, the file you import is `artifacts/update-site-repository/site.xml`.

如何更新 `updatesite.nsf`How To Update `updatesite.nsf`

  1. 打開 Domino 的 `updatesite.nsf`。Open the Domino `updatesite.nsf`.
  2. 選擇 `Import Local Update Site`。Choose `Import Local Update Site`.
  3. 指到解壓後的 `site.xml`。Point it to the extracted `site.xml`.
  4. 匯入完成後，視情況移除舊版 `hello-webapp` feature/plugin。After import, remove older `hello-webapp` feature/plugin entries if needed.
  5. 在 Domino console 執行 `restart task http`。Run `restart task http` in the Domino console.
